Search PRIDE Archive

Previous | Next | See Also

This section explains how to search PRIDE Archive, which offer a list of publicly available shortgun proteomics datasets.

PRIDE Archive is a proteomics data repository, which is part of the ProteomeXchange consortium that has been set up to provide coordinated submisison of MS proteomics data to the main existing proteomics repositories, and to encourage optimal data dissemination.

The public databsets from PRIDE Archive are retrieved through its web service.


You can connect the pride inspector to the public database in 3 different ways:

  • From the Welcome screen, clicking on the Search PRIDE Archive icon
  • From the File | Search PRIDE option in the menu bar
  • From the toolbar, clicking on the button

Once you are entered "Search PRIDE Archive", a search panel consists of three parts will be displayed:
  • Search Box: In this box, you can specify which metadata field you want to search for. And the criteria for searching, as well as the search string. Once the search button is clicked, the table below will show all the matching experiments.
  • Project Table: This table contains all the matching projects from the search, if no search has been made, all the public projects are listed by default. The table also provides some key information about the projects, such as: project title, species, tissue and etc. Users can choose to download all the files belongs to the project by clicking on the download button.
  • Assay Table: You can choose a project of interest from the Project table. If the project is a complete submission, which means it included the experimental results in standard file format. A list of assays will be shown in this table. Users can download specific assay files by clicking on the download button in the relevant row.
