What is peak file Format?

Previous | Next | See Also

Peak files are the spectra files generated after processing the original RAW data from mass spectrometers. This processing step includes peak picking, deconvolution, etc. The most common peaks file formats are: - mgf - ms2 - mzXML - mzData - dta - pkl
