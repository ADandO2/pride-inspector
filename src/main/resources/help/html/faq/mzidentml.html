What is mzIdentML Format?

Previous | Next | See Also

MzIdentML is a data exchange standard for the analysis of proteomic data developed by the Proteomics Information Group (PSI-PI) intended to encapsulate all important information about searches done against MS, MS-MS, MSn data. Enough information is included to repeat the search and examine the results among other capabilities.

The spectra data related with the identifications in the mzIdentML files should be referenced from the mzIdentML files. Currently, PRIDE Inspector supports mzML, mzXML, pkl, mgf, ms2 and dta file formats. The reference to the original spectra must be added during the loading process of each mzIdentML file (More information). If you have problems with spectra referencing please contact the pride support team: PRIDE HelpDesk.
